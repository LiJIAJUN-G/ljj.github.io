conda基本安装与使用

Li jiajun 李嘉俊
9月 17, 2023
conda

conda

基本安装

清华源下载地址https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/

是一个.sh文件

命令行输入sh xxxx.sh进行安装

中间enter+yes，有一步需要设置路径，最后一步问是否自动配置环境变量，可根据自己情况自己配置。

基本操作

创建新环境


conda create -n python python=3 r-base=3.6

创建一个name为python，默认python版本为3的环境

激活conda activate python 退出环境 conda deactivate

pip换源


pip config set global.index-url http://mirrors.aliyun.com/pypi/simple/
pip config set global.trusted-host mirrors.aliyun.com

常用命令

Nohup      #命令 > 1.log &挂在后台运行
Jobs       #查看当前进程
Ps -aux    #查看所有进程
Top        #查看进程
Kill -9    #杀死进程
